Example 2.4.12. Evaluating a Limit by Factoring and Canceling.

Evaluate \(\limit_{x \to 3 }\frac{x^2-3x}{2x^2-5x-3}.\)
🔗
Solution.
  1. The function \(f(x)=\frac{x^2-3x}{2x^2-5x-3}\) is undefined for \(x=3.\) In fact, if we substitute 3 into the function we get \(0/0,\) which is undefined. Factoring and canceling is a good strategy:
    🔗
    \begin{equation*} \limit_{x \to 3 }\frac{x^2-3x}{2x^2-5x-3}=\limit_{x \to 3 }\frac{x(x-3)}{(x-3)}{(2x+1)} \end{equation*}
    🔗
  2. For all \(x\neq 3,\frac{x^2-3x}{2x^2-5x-3}=\frac{x}{2x+1}.\) Therefore,
    🔗
    \begin{equation*} \limit_{x \to 3 }\frac{x(x-3)}{(x-3)}{(2x+1)}=\limit_{x \to 3 }\frac{x}{2x+1}. \end{equation*}
    🔗
  3. Evaluate using the limit laws:
    🔗
    \begin{equation*} \limit_{x \to 3 }\frac{x}{2x+1}=\frac{3}{7}. \end{equation*}
    🔗
