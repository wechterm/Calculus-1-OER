Definition 2.3.1.

Let \(f(x)\) be defined for all \(x\ne a\) over an open interval containing \(a\text{.}\) Let \(L\) be a real number. Then
\begin{equation*} \lim_{x\to a} f(x)=L \end{equation*}
if, for every \(\varepsilon\gt 0\) there exists a \(\delta\gt 0\text{,}\) such that if \(0\lt|x-a|\lt\delta\text{,}\) then \(|f(x)-L|\lt\varepsilon\text{.}\)
