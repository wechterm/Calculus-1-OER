Example 2.1.12. Evaluating a Limit Using a Graph.

For \(g(x)\) shown in Figure 2.1.13, evaluate \(\ds\lim_{x\to-1}g(x).\)
🔗
Solution.
Despite the fact that \(g(-1)=4,\) as the \(x\)-values approach \(-1\) from either side, the \(g(x)\) values approach \(3\text{.}\) Therefore, \(\ds\lim_{x\to-1}g(x)=3.\) Note that we can determine this limit without even knowing the algebraic expression of the function.
