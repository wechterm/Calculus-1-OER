Example 2.4.20. Evaluating a One-Sided Limit Using the Limit Laws.

Evaluate each of the following limits, if possible.
🔗
  1. \(\displaystyle \limit_{x \to 3^-}\sqrt{x-3}\)
    🔗
  2. \(\displaystyle \limit_{x \to 3^+}\sqrt{x-3}\)
    🔗
Solution.
Figure 2.4.21 illustrates the function \(f(x)=\sqrt{x-3}\) and aids in our understanding of these limits.
🔗
  1. The function \(f(x)=\sqrt{x-3}\) is defined over the interval \([3,+\infty).\) Since this function is not defined to the left of 3, we cannot apply the limit laws to compute \(\limit_{x \to 3^-}\sqrt{x-3}.\) In fact, since \(f(x)=\sqrt{x-3}\) is undefined to the left of 3, \(\limit_{x \to 3^-}\sqrt{x-3}\) does not exist.
    🔗
  2. Since \(f(x)=\sqrt{x-3}\) is defined to the right of 3, the limit laws do apply to \(\limit_{x \to 3^+}\sqrt{x-3}.\) By applying these limit laws we obtain \(\limit_{x \to 3^+}\sqrt{x-3}=0.\)
    🔗
