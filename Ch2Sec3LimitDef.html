    Section 2.2 Formal \(\delta-\varepsilon\) Definition of a Limit

    Learning Objectives.

    1. Describe the \(\delta-\varepsilon\) definition of a limit.
      🔗
    2. Apply the \(\delta-\varepsilon\) definition to find the limit of a function.
      🔗
    3. Describe the \(\delta-\varepsilon\) definitions of one-sided limits
      🔗
    So far, you’ve built an intuitive understanding of limits using tables, graphs, one-sided behavior, and examples of when limits fail to exist. Now we turn to the formal, precise, mathematical definition of a limit: the \(\delta-\varepsilon\) (“delta epsilon”) definition. This is often one of the most challenging definitions in early calculus but learning it pays off. Understanding this definition is the key that opens the door to a better understanding of calculus.
    🔗

    Subsection 2.2.1 The Core Idea: Quantifying Closeness

    The intuitive meaning of a limit is
    \begin{equation*} \text{As }x \text{ gets close to }a\text{, the value of }f(x)\text{ gets close to }L. \end{equation*}
    🔗
    We translate what "gets close" means in mathematical language.
    🔗

    Describing closeness in \(x\).

    Saying that "\(x\) is within \(\delta\) units of \(a\)" means \(x\) is a solution to the inequality
    \begin{equation*} |x-a|\lt\delta \end{equation*}
    This simply means the distance between \(x\) and \(a\) is less than \(\delta\)
    🔗
    Because the limit can ignore the value of \(f(x)\) at \(x=a\) (see Example 2.1.12), we can exclude it from the inequality by specifying
    \begin{equation*} 0\lt|x-a|\lt \delta \end{equation*}
    This means the distance between \(x\) and \(a\) is less than \(\delta\) and \(x\ne a\)
    🔗
    🔗

    Describing closeness in \(y\).

    Similarly, "\(f(x)\) is within \(\varepsilon\) units of \(L\)" can be written
    \begin{equation*} |f(x)-L|\lt \varepsilon \end{equation*}
    This means the distance between the \(y\)-values of the function and the limit is less than \(\varepsilon\)
    🔗
    🔗

    Putting these ideas together.

    The formal definition rewrites the informal statement
    \begin{equation*} \text{As }x \text{ gets close to }a\text{, the value of }f(x)\text{ gets close to }L. \end{equation*}
    as
    \begin{equation*} \text{If }0\lt|x-a|\lt\delta\text{, then }|f(x)-L|\lt \varepsilon \end{equation*}
    This is the heart of the \(\delta-\varepsilon\) definition.
    🔗
    🔗

    Definition 2.2.1.

    Let \(f(x)\) be defined for all \(x\ne a\) over an open interval containing \(a\text{.}\) Let \(L\) be a real number. Then
    \begin{equation*} \lim_{x\to a} f(x)=L \end{equation*}
    if, for every \(\varepsilon\gt 0\) there exists a \(\delta\gt 0\text{,}\) such that if \(0\lt|x-a|\lt\delta\text{,}\) then \(|f(x)-L|\lt\varepsilon\text{.}\)
    🔗
    🔗
    Let’s look at Table 2.2.2, which breaks down the definition and translates each part.
    🔗
    We can get a better handle on this definition by looking at the definition geometrically. Figure 2.2.3 shows possible values of \(\delta\) for various choices of \(\varepsilon\) for a given function \(f(x)\text{,}\) a number \(a\text{,}\) and a limit \(L\text{.}\)
    🔗
    Notice that as we choose smaller values of \(\varepsilon\) (the distance between the function and the limit), we can always find a \(\delta\) small enough so that we have chosen an \(x\) value within \(\delta\) of \(a\text{,}\) then the value of \(f(x)\) is within \(\varepsilon\) of the limit \(L\text{.}\)
    🔗

    Insight 2.2.4.

    Example 2.2.5. Proving a Statement About the Limit of a Specific Function.

    Prove that \(\ds\lim_{x\to 1}(2x+1)=3\)
    🔗
    Answer.
    Let \(\varepsilon\gt 0\text{.}\)
    🔗
    Choose \(\delta=\varepsilon/2\text{.}\)
    🔗
    Assume \(0\lt|x-1|\lt\delta\text{.}\)
    🔗
    Thus,
    \begin{align*} |(2x+1)-3|\amp=|2x-2|\\ \amp=|2(x-1)|\\ \amp=|2||x-1|\\ \amp=2|x-1|\\ \amp\lt 2\delta\\\ \amp=2\cdot\frac{\varepsilon}{2}\\ \amp=\varepsilon \end{align*}
    🔗
    Therefore, \(\ds\lim_{x\to 1}(2x+1)=3\)
    🔗
    🔗
    Solution.
    We started with \(|(2x+1)-3|\) and used our assumption \(0\lt|x-1|\lt\delta\) in a key part of the chain of inequalities to get \(|(2x+1)-3|\) to be less than \(\varepsilon\text{.}\)
    🔗
    We could just as easily have manipulated the assumed inequality \(0\le|x-1|\lt \delta\) to arrive at \(|(2x+1)-3|\lt\varepsilon\) as follows:
    \begin{align*} 0\lt|x-1|\lt\delta \amp \Rightarrow|x-1|\lt\delta\\ \amp \Rightarrow -\delta\lt x-1\lt\delta\\ \amp \Rightarrow -\frac{\varepsilon}{2}\lt x-1\lt\frac{\varepsilon}{2}\\ \amp \Rightarrow -\varepsilon\lt 2x-2\lt \varepsilon\\ \amp \Rightarrow |2x-2|\lt\varepsilon\\ \amp \Rightarrow |(2x+1)-3|\lt\varepsilon \end{align*}
    🔗
    Therefore, \(\ds\lim_{x\to1}(2x+1)=3\text{.}\) (Having completed the proof, we state what we have accomplished.)
    🔗
    🔗
    🔗

    Note 2.2.6. Problem-Solving Strategy: Proving that \(\ds \lim_{x\to a}f(x)=L\) for a Specific Function \(f(x)\).

    1. Let’s begin the proof with the following statement: Let \(\varepsilon\gt 0\text{.}\)
      🔗
      🔗
    2. Next we need to obtain a value for \(\delta\text{.}\) After we have obtained this value, we make the following statement, filling in the blank with our choice of \(\delta\text{:}\) Choose \(\delta=\rule{20pt}{.5pt}\)
      🔗
      🔗
    3. The next statement in the proof should be (at this point, we fill in our given value for \(a\)): Assume \(0\lt |x-a\lt\delta\text{.}\)
      🔗
      🔗
    4. Next, based on this assumption, we need to show that \(|f(x)-L|\lt \varepsilon\text{,}\) where \(f(x)\) and \(L\) are our function \(f(x)\) and our limit \(L\text{.}\) At some point, we need to use \(0\lt |x-a|\lt \delta\text{.}\)
      🔗
      🔗
    5. We conclude our proof with the statement: Therefore, \(\ds \lim_{x\to a}f(x)=L\text{.}\)
      🔗
      🔗
    🔗

    Example 2.2.7. Proving a Statement about a Limit.

    Complete the proof that \(\ds\lim_{x\to -1}(4x+1)=-3\) by filling in the blanks.
    🔗
    Let \(\rule{20pt}{.5pt}\text{.}\)
    🔗
    Choose \(\delta=\rule{20pt}{.5pt}\text{.}\)
    🔗
    Assume \(0\lt |x-\rule{20pt}{.5pt}|\lt \delta\text{.}\)
    🔗
    Thus, \(|\rule{20pt}{.5pt}-\rule{20pt}{.5pt}|\lt\rule{20pt}{.5pt}=\varepsilon\text{.}\)
    🔗
    Solution.
    We begin by filling in the blanks where the choices are specified by the definition. Thus we have
    🔗
    Let \(\varepsilon\gt 0\text{.}\)
    🔗
    Choose \(\delta=\rule{20pt}{.5pt}\text{.}\)
    🔗
    Assume \(0\lt |x-(-1)|\lt \delta\) (or equivalently, \(0\lt |x+1|\lt \delta\)).
    🔗
    Thus, \(|(4x+1)-(-3)|=|4x+4|=|4||x+1|\lt 4\delta\rule{20pt}{.5pt}\varepsilon\text{.}\)
    🔗
    Focusing on the final line of the proof, we see that we should choose \(\delta=\frac{\varepsilon}{4}\text{.}\)
    🔗
    We now complete the final write-up of the proof:
    🔗
    Let \(\varepsilon\gt 0\text{.}\)
    🔗
    Choose \(\delta=\frac{\varepsilon}{4}\text{.}\)
    🔗
    Assume \(0\lt |x-(-1)|\lt \delta\) (or equivalently, \(0\lt|x+1|\lt\delta\)).
    🔗
    Thus, \(|(4x+1)-(-3)|=|4x+4|=|4||x+1|\lt 4\delta=4(\varepsilon/4)=\varepsilon\)
    🔗
    🔗
    🔗

    Example 2.2.8. Proving a Statement about the Limit of a Specific Function (Geometric Approach).

    Prove that \(\ds\lim_{x\to 2}x^2=4\)
    🔗
    Solution.
    1. Let \(\varepsilon\gt 0\text{.}\) The first part of the definition begins “For every \(\varepsilon\gt 0\text{,}\)” so we must prove that whatever follows is true no matter what positive value of \(\varepsilon\) is chosen. By stating “Let \(\varepsilon\gt 0\text{,}\)” we signal our intent to do so.
      🔗
      🔗
    2. Without loss of generality, assume \(\varepsilon\le 4\text{.}\) Two questions present themselves: Why do we want \(\varepsilon\le 4\) and why is it okay to make this assumption?
      🔗
      In answer to the first question: Later on, in the process of solving for \(\delta\text{,}\) we will discover that \(\delta\) involves the quantity \(\sqrt{4-\varepsilon}\text{.}\) Consequently, we need \(\varepsilon\le 4\text{.}\)
      🔗
      In answer to the second question: If we can find \(\delta\gt 0\) that “works” for \(\varepsilon\le 4\text{,}\) then it will “work” for any \(\varepsilon \gt 4\) as well. Keep in mind that, although it is always okay to put an upper bound on \(\varepsilon\text{,}\) it is never okay to put a lower bound (other than zero) on \(\varepsilon\)
      🔗
      🔗
    3. Choose \(\delta=\min\left\{2-\sqrt{4-\varepsilon}, \sqrt{4+\varepsilon}-2\right\}\text{.}\) Figure 2.2.9 shows how we made this choice of \(\delta\text{.}\)
      🔗
      🔗
    4. We must show: If \(0\lt|x-2|\lt\delta\text{,}\) then \(|x^2-4|\lt \varepsilon\text{,}\) so we must begin by assuming
      \begin{equation*} 0\lt|x-2|\lt \delta \end{equation*}
      We don’t really need \(0\lt|x-2|\) (in other words, \(x\ne 2\)) for this proof. Since \(0\lt |x-2|\lt \delta\Rightarrow|x-2|\lt \delta\text{,}\) it is okay to drop \(0\lt |x-2|\text{.}\)
      \begin{equation*} |x-2|\lt \delta \end{equation*}
      Hence,
      \begin{equation*} -\delta\lt x-2\lt\delta \end{equation*}
      🔗
      Recall that \(\delta=\min\left\{2-\sqrt{4-\varepsilon}, \sqrt{4+\varepsilon}-2\right\}\text{.}\) Thus, \(\delta\le 2-\sqrt{4-\varepsilon}\) and consequently \(-(2-\sqrt{4-\varepsilon})\le -\delta\text{.}\) We also use \(\delta\le \sqrt{4+\varepsilon}-2\) here.
      🔗
      We might ask at this point: Why did we substitute \(2-\sqrt{4-\varepsilon}\) for \(\delta\) on the left-hand side of the inequality and \(\sqrt{4+\varepsilon}-2\) on the right-hand side of the inequality? If we look at Figure 2.2.9, we see that \(2-\sqrt{4-\varepsilon}\) corresponds to the distance on the left of \(2\) on the \(x\)-axis and \(\sqrt{4+\varepsilon}-2\) corresponds to the distance on the right. Thus,
      \begin{equation*} -(2-\sqrt{4-\varepsilon})\le -\delta\lt x-2\lt\delta\le\sqrt{4+\varepsilon}-2 \end{equation*}
      🔗
      We simplify the expression on the left:
      \begin{equation*} -2+\sqrt{4-\varepsilon}\lt x-2\lt \sqrt{4+\varepsilon}-2 \end{equation*}
      Then, we add \(2\) to all parts of the inequality:
      \begin{equation*} \sqrt{4-\varepsilon}\lt x\lt\sqrt{4+\varepsilon} \end{equation*}
      We square all parts of the inequality. It is okay to do so, since all parts of the inequality are positive:
      \begin{equation*} 4-\varepsilon\lt x^2\lt 4+\varepsilon \end{equation*}
      We subtract \(4\) from all parts of the inequality:
      \begin{equation*} -\varepsilon\lt x^2-4\lt \varepsilon \end{equation*}
      Last,
      \begin{equation*} |x^2-4|\lt \varepsilon \end{equation*}
      🔗
      🔗
    5. Therefore,
      \begin{equation*} \lim_{x\to 2}x^2=4 \end{equation*}
      🔗
      🔗
    🔗
    🔗

    Example 2.2.10. Proving a Statement about the Nonexistence of a Limit.

    Let
    \begin{equation*} f(x)=\begin{cases}-1,& x\lt 0\\1,&x\ge 0\end{cases} \end{equation*}
    Prove that \(\ds\lim_{x\to 0}f(x)\) does not exist.
    🔗
    Solution.
    One way to prove that a limit does not exist is to show that, for any possible value \(L\) one might think the limit equals, one can find an \(\varepsilon\gt 0\) such that no possible \(\delta\gt 0\) works in the \(\varepsilon-\delta\) definition.
    🔗
    Suppose a person think the limit exists, and \(\ds\lim_{x\to 0}f(x)=L\) (some common values of \(L\) might be \(1\) or \(-1\text{,}\) but we’ll stick with a generic real number). Let \(\varepsilon=0.5\text{.}\) We show that there is no possible \(\delta\gt 0\) that would satisfy Definition 2.2.1.
    🔗
    The inequality \(|f(x)-L|\lt 0.5\) can be written as
    \begin{equation*} L-.5\lt f(x)\lt L+0.5 \end{equation*}
    For any \(\delta\gt 0\text{,}\) the solution to \(|x-0|\lt \delta\) are \(x-\)values \(-\delta\lt x\lt\delta\text{.}\) This interval contains both positive and negative numbers. When \(x\) is positive, \(f(x)=1\) and when \(x\) is negative, \(f(x)=-1\text{.}\) Since these two values of \(f(x)\) are \(2\) units, apart, it would be impossible for them both to be within \(0.5\) of any number \(L\text{.}\)
    🔗
    That is, if
    \begin{equation*} L-0.5\lt -1\lt L+0.5 \end{equation*}
    then it can’t also be true that
    \begin{equation*} L-0.5\lt 1\lt L+0.5, \end{equation*}
    and vice-versa. Any \(\delta\gt 0\) will result in an interval with both positive and negative \(x-\)values, thus no value of \(\delta\gt 0\) can satisfy the \(\delta-\varepsilon\) defintion.
    🔗
    🔗
    🔗
    🔗

    Subsection 2.2.2 One-Sided Limits

    Just as we first gained an intuitive understanding of limits and then moved on to a more rigorous definition of limits, we now revisit one-sided limits through that same lens. To do this, we modify the \(\delta-\varepsilon\) definition of a limit to give formal \(\delta-\varepsilon\) definitions for limits from the left and from the right at a point.
    🔗
    These definitions require only slight modifications from the definition of the limit. In the definition of the limit from the right, the inequality \(0\lt x-a\lt \delta\) replaces \(0\lt |x-a|\lt \delta\text{,}\) which ensures that we only consider values of \(x\) that are greater than (to the right of) \(a\text{.}\)
    🔗
    Similarly, in the definition of the limit from the left, the inequality \(-\delta\lt x-a\le 0\) replaces \(0\lt|x-a|\lt \delta\text{,}\) which ensures that we only consider values of \(x\) that are less than (to the left of) \(a\text{.}\)
    🔗

    Definition 2.2.11.

    Limit from the Right:.
    Let \(f(x)\) be defined over an open interval of the form \((a,b)\) where \(a\lt b\text{.}\) Then,
    \begin{equation*} \lim_{x\to a^+}f(x)=L \end{equation*}
    if for every \(\varepsilon\gt 0\text{,}\) there exists a \(\delta\gt 0\) such that if \(0\lt x-a\lt \delta\text{,}\) then \(|f(x)-L|\lt\varepsilon\)
    🔗
    🔗
    Limit from the Left.
    Let \(f(x)\) be defined over an open interval of the form \((b,a)\) where \(b\lt a\text{.}\) Then,
    \begin{equation*} \lim_{x\to a^-}f(x)=L \end{equation*}
    if for every \(\varepsilon\gt 0\text{,}\) there exists a \(\delta\gt 0\) such that if \(-\delta\lt x-a\lt 0\text{,}\) then \(|f(x)-L|\lt \varepsilon\text{.}\)
    🔗
    🔗
    🔗

    Example 2.2.12. Proving a Statement about a Limit From the Right.

    Prove that \(\ds\lim_{x\to 4^+}\sqrt{x-4}=0\)
    🔗
    Solution.
    Let \(\varepsilon\gt 0\text{.}\)
    🔗
    Choose \(\delta=\varepsilon^2\text{.}\) Since we ultimately want \(|\sqrt{x-4}-0|\lt \varepsilon\text{,}\) we manipulate this inequality to get \(\sqrt{x-4}\lt \varepsilon\text{,}\) or equivalently, \(0\lt x-4\lt \varepsilon^2\text{,}\) making \(\delta=\varepsilon^2\) a clear choice.
    🔗
    We may also determine \(\delta\) geometrically as shown in Figure 2.2.13
    🔗
    Assume \(0\lt x-4\lt \delta\text{.}\) Thus, \(0\lt x-4\lt \varepsilon^2\text{.}\) Hence, \(0\lt \sqrt{x-4}\lt \varepsilon\text{.}\) Finally, \(|\sqrt{x-4}-0|\lt \varepsilon\text{.}\)
    🔗
    Therefore, \(\ds\lim_{x\to 4^+}\sqrt{x-4}=0\text{.}\)
    🔗
    🔗
    🔗
    We do not normally use the \(\delta-\varepsilon\) definition to compute everyday limits like the ones you’ve seen earlier. Instead, the definition is used to prove the limit laws you’ll see in the next section. Once these theorems are proven, you can use them freely—without going back to \(\delta\) and \(\varepsilon\) each time.
