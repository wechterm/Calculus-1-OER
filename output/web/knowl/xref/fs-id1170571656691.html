Example 1.1.6. Evaluating a Limit Using a Table of Functional Values 2.

Evaluate \(\lim_{x\to4}\frac{\sqrt{x}-2}{x-4}\) using a table of functional values.
🔗
Solution.
As before, we use a table—in this case, Table 1.1.7—to list the values of the function for the given values of \(x\text{.}\)
🔗
After inspecting this table, we see that the functional values less than 4 appear to be decreasing toward 0.25 whereas the functional values greater than 4 appear to be increasing toward 0.25. We conclude that \(\lim_{x\to4} \frac{\sqrt{x}-2}{x-4}=0.25.\) We confirm this estimate using the graph of \(f(x)=\frac{\sqrt{x}-2}{x-4}\) shown in Figure 1.1.8.
